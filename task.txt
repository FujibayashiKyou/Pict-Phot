Вот тестовое задание.
Надо создать не большой графический редактор по аналогии с приложением под 
iphone Multiexpo  https://itunes.apple.com/ru/app/multiexpo/id600476635?mt=8 . 
Надо реализовать первый и второй скрин (не полный функционал).
Первый скрин (выбор картинок):

-          На нем отображаем два окошка - кастомный компонент (это обычный Layout c ImageView(PhotoView) и bacground в виде фоторамки. 
Можно background выбрать на свое усмотрение). Рамки обязательно должны быть развернуты как на примере. 
Чтобы не было неровностей при повороте компонента, надо по краям в background  убрать один пиксель;

-          При щелчке на окошке, нужно отобразить выпадающие меню в стиле  Path 2.0, можно ArcMenu https://github.com/daCapricorn/ArcMenu. 
При щелчке на пункте меню, оно должно спрятаться  за окошко;

-          Меню состоит из двух пунктов: 

-              “Галерея”  Здесь нужно выполнить обычный интент для выбора картинки с галереи.  
В окошко вставляем картинку в ужатом виде (thumbnail), а оригинал сохраняем для дальнейшей обработки. 
Также нужно предусмотреть загрузку больших картинок (не должна превышать  размер 2000x2000. Могу скинуть либу для сжатия);

-              “Instagram” Выводим новое активити с фотками  с Instagram.  Можно использовать разметку GridLayout. 
При щелчке на фотку, оно вставляется в рамочку. Все остальное по аналогии с галереей.  При вставке можно использовать анимацию затухания.

-       Когда все будет выбрано, должна активироваться кнопка «Обработать”.

Второй скрин (обработка картинок):
-            Появится фрагмент для наложения и обрезки фотографий.  В нижней части будут наши рамочки с миниатюрами фоток с первого фрагмента.  
Центральную часть окна будут занимать наложенные рисунки с рамочкой для обрезки.  Для обрезки можно использовать эту библиотеку https://github.com/edmodo/cropper ;

-       Результат сохранить в галерею или Instagram.

Api  4.1.x – высше
